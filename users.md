# 用户 API

## 获取用户

<table>
  <tr>
    <th align="right">请求</th>
    <td><code>GET /user/:uid</code></td>
  </tr>
  <tr>
    <th align="right">参数</th>
    <td><code>{ _contentOnly: any; }</code></td>
  </tr>
  <tr>
    <th align="right">响应主体</th>
    <td><code>application/json</code> (<code>DataResponse&lt;UserData&gt;</code>)</td>
  </tr>
</table>
